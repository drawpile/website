{% filter commonmark_full %} ## Get Help, Give Suggestions, Report Bugs
 Discord  Web Chat [m] Matrix  GitHub
Joining the **Drawpile Discord Server** is probably the easiest way to get help, since there's a number of nice people at the ready to answer questions and provide assistance. It's also a good place to keep up with the latest development, provide suggestions, report issues and so on. You can also jump directly into Drawpile's IRC chatroom using **the web chat**. Just press the "start" button and talk away, you don't need an account. If you already have an IRC client, you can join the *#drawpile* channel on libera.chat. It's connected to the Discord server, so people from there will also be able to help you. If you use Matrix, there's also **the #drawpile:matrix.org room** on there. It is also connected to Discord, so folks there will see your messages. You can also report stuff in the **GitHub Issue Tracker** directly if you like.
